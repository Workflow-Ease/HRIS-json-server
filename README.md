## json-server

### install 
```bash
$ npm install json-server
```

### Usage
- create employee.json

### Pass it to JSON Server CLI
```bash
**$ npx json-server employee.json**

JSON Server started on PORT :3000
Press CTRL-C to stop

http://localhost:3000/

http://localhost:3000/departmentalSales
```
